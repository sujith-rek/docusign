# Emoji Viewer

## Deployment
#### Install dependencies
```bash
npm install
```
#### Run the app
```bash
ng serve
```
and go to `http://localhost:4200/`


## Made with
- [Angular](https://angular.io/)
- [Angular Material](https://material.angular.io/)
- [Angular Bootstrap](https://ng-bootstrap.github.io/)
